# Portfolio Website

## Run Locally

`Clone the project -- git clone https://github.com/Zarfaa/Portfolio`<br><br>

`Go to the project directory: cd Portfolio`<br><br>

`Install dependencies: npm install`<br><br>

`Start the server: npm start`<br><be>
